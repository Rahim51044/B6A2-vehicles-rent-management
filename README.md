## Project Description
This is a backend REST API for a Vehicle Rental Management System.
It supports vehicle management, user authentication, role-based access control,
and booking management with automatic price calculation.

##Features
- JWT Authentication (Admin & Customer)
- Role-based Authorization
- Vehicle CRUD Operations (Admin only)
- Booking Management
  - Create booking
  - View bookings (Admin: all, Customer: own)
  - Cancel / Return booking
- Automatic vehicle availability update
- PostgreSQL database with relations
